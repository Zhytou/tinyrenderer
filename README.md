**Build & Run**:

```bash
# build and compile
mkdir build
cd build
cmake -S .. -B .
make all

# run
./main
```

**Main Features**:

- .obj loading.
- physically based rendering.
- basic altas shadow mapping.
- skybox rendering.
- image based lighting.
